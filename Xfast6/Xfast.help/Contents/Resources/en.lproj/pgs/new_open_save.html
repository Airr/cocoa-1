New, Open, Save .kmr File

Alder Kmer provides you with a simple way of saving the current run, and opening a run. The file name extension is .kmr.

  • Use File>New menu to initialize the window of Alder Kmer.

  • Use File>Open menu to open a run file.

  • Use File>Save menu to save the current run.

NOTE: These menus are inactivated during the operation of counting Kmers.
