Alder Kmer Help

About Alder Kmer

Count fixed-length DNA sequences using multiple CPUs and large disk space with limited memory, query a fixed-length DNA sequence for the occurence, create runs of counting.

How to Run

Load a sequence file, configure a run, and start counting fixed-length DNA sequences in the sequence file.

New, Open, and Save

Save a run, and open it later.

Supported Sequence File Formats

FASTQ and FASTA with and without gzipped.
