About Alder Kmer

With Alder Kmer, you can count fixed-length DNA sequences or Kmers in sequence files with your computers. Personal computers with multiple CPUs and relative large disk space would benefit from using Alder Kmer. You can adjust the maximum allowed disk space, memory, and CPU cores available to Alder Kmer. Set the size of Kmers, and output directory with out filename prefix. Then, start counting Kmers. After a run of counting Kmers finished, you would see the summary of the analysis. Query a Kmer for the number of the occurences in the input data file. You can also save a run in a .kmr file for later rerun.
